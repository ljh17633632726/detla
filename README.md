# 三角洲护航小程序

#### 介绍

三角洲护航平台，使用 Spring Boot + Vue + UniApp 开发，支持微信小程序，支持唤起微信支付。

技术栈：**Spring Boot**、**Vue 3**、**UniApp**、**微信小程序**、**微信支付**。

#### 功能说明

三角洲护航点单成品微信小程序，适用于俱乐部、和平精英、暗区突围、英雄联盟等护航陪玩场景。

**端结构**
- 原生微信小程序 + PC 后台
- PC 端：管理员 + 多客服管理
- 小程序端：用户端 + 接单端（打手端）+ 客服端

**功能点**  
一、核心流程功能<br>
下单：用户在小程序内选择服务并提交订单。<br>
派单：支持客服派单、自动派单、指定单、抢单、组队单。<br>
接单：打手在打手端确认接单并开始服务。<br>
进度：打手更新服务进度，用户可实时查看。<br>
完成：打手标记服务完成，系统通知用户。<br>
评价 / 投诉：用户对订单进行评价或发起投诉，客服介入处理。<br>
换人申请：服务过程中客户可发起换人申请，由客服审核后更换打手并调整订单归属。<br>
打手换人申请：换自己，换队友，两个都换<br>
a. 两个都换 - 主接单员+队友都退出，订单放回接单大厅，双方都不拿钱<br>
b. 换队友 - 踢掉队友（队友不参与分成），自己继续做，可以重新邀请队友并选择新的分成方式<br>
<br>
二、聊天与客服功能<br>
打手聊天：支持文本等基础消息。<br>
聊天卡片：支持在聊天中发送订单卡片、商品卡片。<br>
手机客服：客服可用手机登录小程序接待咨询、派单、转单、引导下单。<br>
<br>
三、订单与运营功能<br>
多种派单方式：客服派单、自动派单、指定单、抢单、组队单。<br>
转单：已接订单支持转交其他打手。<br>
下单提醒：未支付订单和咨询用户可推送下单/支付提醒。<br>
热门分类：支持配置热门分类，在前端优先展示。<br>
热门分类管理：支持在后台新增、编辑、排序和上下架分类，可设置热门标签并按业务数据动态调整。<br>
动态表单：下单表单项可根据商品所属父分类动态变化，支持不同业务场景下的字段配置。<br>
消费榜单：支持按用户消费金额、次数等维度生成消费榜单，用于活动与运营展示。<br>
商品限购：支持为商品开启每人限购开关，配置单用户最多购买次数，并在小程序商品详情展示“限购 N 次”提示分为每周限购，每月限购，永久限购。<br>
组队邀请时效：打手邀请队友后 5 分钟内未接受则自动失效，并通知主打手可重新邀请。<br>
<br>
四、支付与结算功能<br>
微信支付：唤起微信支付，支付成功立即到账。<br>
自定义抽成：平台抽佣比例支持 0%–50% 自由设置，立即生效。<br>
打手提现：打手余额结算后可发起提现申请。<br>
退款：支持订单退款 / 部分退款，金额原路退回。<br>
打手注册押金：支持设置打手入驻押金标准，押金支付后方可接单，可用于违约或处罚时扣除。<br>
商品级抽成：支持为单个商品设置独立抽成比例，如已设置则优先按商品抽成比例结算，否则按平台默认抽成比例。<br>
<br>
五、风控与管理功能<br>
投诉管理：用户投诉由客服受理并处理。<br>
打手处罚：支持禁接单 禁止提现处罚方式，与投诉记录联动。<br>
通知提醒：客户可收到被接单、服务完成、退款等提醒；打手可收到新订单、超时、提现等提醒<br>

#### 部署说明

1. 详见仓库内 **部署文档.html**（服务器部署、Nginx 配置、Java 启动）
2. **服务器搭建.html**：购买服务器、域名、SSL 证书
3. **小程序AppID与商户密钥获取.html**：微信 AppID、商户号、支付密钥、订阅消息

#### 安装教程

1. 克隆本仓库（或直接下载上方「打包项目压缩包」并解压）
2. 将 `delta-admin.jar` 上传到服务器（如 `/www/wwwroot/delta-admin.jar`）
3. 按 **部署文档.html** 完成服务器环境、Nginx、MySQL、Redis 配置
4. 使用微信开发者工具打开 `mp-weixin` 目录，配置域名后上传小程序
