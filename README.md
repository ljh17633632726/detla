# 护航小程序

护航小程序是一套面向电竞陪玩、游戏代练、护航服务、虚拟服务接单场景的成品系统。项目包含用户端、接单员端、手机客服端和 PC 管理后台，覆盖用户下单、在线支付、客服派单、接单员服务、进度跟踪、评价投诉、退款售后、收益结算和提现审核等完整流程。

技术栈：Spring Boot、Vue 3、UniApp、微信小程序、微信支付、MySQL、Redis。

> 本仓库提供的是可部署的打包交付物。完整源码属于收费交付内容，如需源码请按下方“源码获取”说明购买卡密并完成校验。

## 项目介绍

本系统适合用于搭建电竞陪玩、游戏代练、俱乐部护航、虚拟商品服务、同城/线上接单等平台。用户可以在小程序内浏览服务、提交订单、在线支付、查看进度并与客服或接单员沟通；接单员可以入驻、抢单、接单、组队、提交进度、查看收益并申请提现；客服和管理员可以在手机端或 PC 后台处理派单、退款、投诉、换人、提现和商品运营。

README 只保留基础介绍和快速入口。更详细的项目说明、功能清单和部署教程，请查看 `文档/` 目录中的内容。

## 核心能力

- 多端完整：用户小程序、接单员端、手机客服端、PC 管理后台。
- 订单闭环：下单、支付、派单、接单、进度、完成、评价、投诉、退款。
- 接单模式：客服派单、指定接单员、抢单、组队单、接力、换人。
- 运营管理：商品、分类、热门分类、动态下单字段、公告、轮播、快捷回复。
- 资金结算：微信支付、余额支付、退款、平台抽成、商品抽成、组队分成、提现审核。
- 风控配置：接单员押金、在线状态、并发接单上限、投诉处罚、禁止接单、禁止提现。
- 消息提醒：站内消息、聊天会话、订单卡片、商品卡片、阿里云短信提醒。

## 部署简要

1. 导入 `delta_game.sql` 初始化数据库，需要测试数据时再导入 `test_seed_data.sql`。
2. 部署后端 jar，并按部署文档配置 MySQL、Redis、微信支付、上传目录等参数。
3. 将 `dist/` 部署到 Nginx 或宝塔站点，作为 PC 管理后台。
4. 使用微信开发者工具打开 `移动端/mp小程序端/`，配置 AppID 和合法域名后上传小程序。
5. H5 端可部署 `移动端/h5端/`，并按文档配置接口地址和路由回退。

详细步骤请以 `文档/部署文档.html` 为准。
