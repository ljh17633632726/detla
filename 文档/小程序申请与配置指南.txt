一、小程序注册要求
--------------------------------------------------------------------------------

1. 主体类型：必须使用【企业主体】注册
   - 个人主体无法开通支付功能
   - 个人主体无法获取用户手机号
   - 建议使用公司营业执照注册

2. 注册地址：https://mp.weixin.qq.com/
   - 选择"小程序"进行注册
   - 准备企业营业执照、法人身份证、对公账户

3. 认证费用：300元/年（企业认证）


二、必须开通的权限与接口
--------------------------------------------------------------------------------

【基础权限】（注册后默认开通）
  ✓ wx.login - 微信登录
  ✓ wx.request - 网络请求
  ✓ wx.connectSocket - WebSocket连接
  ✓ wx.chooseImage - 选择图片
  ✓ wx.uploadFile - 上传文件
  ✓ wx.getStorageSync - 本地存储

【需要申请的权限】

1. 获取手机号（getPhoneNumber）
   - 条件：完成企业认证
   - 用途：用户绑定手机号
   - 申请位置：自动开通，无需单独申请

2. 微信支付
   - 条件：需要开通微信支付商户号
   - 申请地址：https://pay.weixin.qq.com/
   - 所需材料：
     * 营业执照
     * 法人身份证
     * 对公银行账户
     * 小程序AppID
   - 费率：一般为0.6%

3. 订阅消息
   - 申请位置：小程序后台 -> 功能 -> 订阅消息
   - 需要申请的模板：
     * 订单支付成功通知
     * 订单状态变更通知
     * 订单完成通知
     * 新订单提醒
     * 提现到账通知
   - 注意：选择与业务相关的类目模板

三、服务类目选择（重要！）
--------------------------------------------------------------------------------

游戏代练平台建议选择以下类目：

1. 主类目：生活服务  >  电子娱乐服务

【注意事项】
- 游戏类目审核严格，可能需要版号
- 避免直接选择"游戏"类目
- 类目决定了可使用的订阅消息模板
- 类目选错可能导致审核不通过

【可能需要的资质】
- 《增值电信业务经营许可证》（ICP证）- 如涉及在线交易
- 软件著作权证书 - 如有自研系统


四、域名配置（必须）
--------------------------------------------------------------------------------

位置：小程序后台 -> 开发管理 -> 开发设置 -> 服务器域名

需要配置的域名：

1. request合法域名（HTTPS接口）
   示例：https://api.yourdomain.com

2. socket合法域名（WebSocket）
   示例：wss://ws.yourdomain.com

3. uploadFile合法域名（文件上传）
   示例：https://upload.yourdomain.com

4. downloadFile合法域名（文件下载）
   示例：https://cdn.yourdomain.com

【域名要求】
- 必须是备案过的域名
- 必须使用HTTPS/WSS协议
- 不能使用IP地址
- 不能使用localhost
- 每月最多修改5次


五、微信支付商户号配置
--------------------------------------------------------------------------------

1. 注册商户号
   地址：https://pay.weixin.qq.com/

2. 关联小程序
   位置：商户平台 -> 产品中心 -> AppID账号管理
   操作：添加小程序AppID并确认关联

3. 配置API密钥
   位置：商户平台 -> 账户中心 -> API安全
   操作：设置APIv3密钥（32位字符串）

4. 下载证书
   位置：商户平台 -> 账户中心 -> API安全
   下载：API证书（用于退款等敏感操作）

六、隐私协议配置（必须）
--------------------------------------------------------------------------------

位置：小程序后台 -> 用户头像 ->账号设置 -> 服务内容声明 -> 用户隐私保护指引

必须填写的内容：
1. 收集的用户信息类型
   - 微信昵称、头像
   - 手机号码
   - 设备信息

2. 信息使用目的
   - 用户身份识别
   - 订单通知
   - 客服沟通

3. 第三方共享说明
   - 支付服务（微信支付）
   - 云存储服务（如有）

【注意】
- 2023年9月起，未配置隐私协议的小程序无法调用隐私接口
- 首次使用需弹出隐私协议弹窗让用户同意


七、开发配置
--------------------------------------------------------------------------------

1. 获取AppID和AppSecret
   位置：小程序后台 -> 开发管理 -> 开发设置

2. 配置开发者
   位置：小程序后台 -> 管理 -> 成员管理
   添加开发者微信号

3. 下载开发者工具
   地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

4. 关闭域名校验（仅开发阶段）
   开发者工具 -> 详情 -> 本地设置 -> 不校验合法域名


八、上线前检查清单
--------------------------------------------------------------------------------

□ 企业认证已完成
□ 微信支付商户号已开通并关联
□ 服务器域名已配置（HTTPS）
□ WebSocket域名已配置（WSS）
□ 隐私协议已填写
□ 用户协议页面已完成
□ 订阅消息模板已申请
□ 服务类目已选择正确
□ 小程序名称、头像、简介已设置
□ 客服功能已配置


九、审核注意事项
--------------------------------------------------------------------------------

1. 首次提审建议
   - 功能完整，无明显BUG
   - 页面不能有空白或"敬请期待"
   - 所有按钮都要有响应

2. 容易被拒的原因
   - 涉及虚拟商品交易未说明
   - 类目与实际功能不符
   - 缺少用户协议/隐私政策
   - 诱导分享或关注
   - 存在敏感词汇

3. 审核时间
   - 一般1-7个工作日
   - 可申请加急审核（有次数限制）

4. 游戏代练特别注意
   - 避免使用"代练"等敏感词作为小程序名称
   - 描述中使用"游戏服务"、"游戏陪玩"等词汇
   - 不要出现具体游戏名称（版权问题）
