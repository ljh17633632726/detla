# 移动端目录说明

本仓库的移动端交付物统一放在 `移动端/` 目录下，包含 H5 端和微信小程序端两套构建产物。

## 目录结构

| 目录 | 说明 |
| --- | --- |
| `移动端/h5端/` | UniApp H5 端打包产物，可部署到 Nginx、宝塔站点或其它静态 Web 服务。入口文件为 `index.html`。 |
| `移动端/mp小程序端/` | 微信小程序端打包产物，使用微信开发者工具打开该目录后上传发布。目录内包含 `project.config.json`。 |
| `mp-weixin/` | 兼容旧版交付结构保留的微信小程序目录，内容与 `移动端/mp小程序端/` 同步。推荐新部署优先使用 `移动端/mp小程序端/`。 |
| `dist/` | PC 管理后台前端打包产物，不属于移动端。 |

## H5 端部署

1. 将 `移动端/h5端/` 目录内的所有文件上传到站点根目录或指定 H5 访问目录。
2. Nginx 站点根目录应指向该目录，入口为 `index.html`。
3. 如果使用子路径部署，需要同步检查前端构建基础路径和 Nginx 路由回退配置。

## 微信小程序端上传

1. 打开微信开发者工具。
2. 选择导入项目，项目目录选择 `移动端/mp小程序端/`。
3. 检查 AppID、request 合法域名、uploadFile/downloadFile 合法域名、业务域名等配置。
4. 预览无误后提交上传并在微信公众平台提交审核。

## 更新说明

后续重新打包移动端时，需要同时更新：

- `移动端/h5端/`
- `移动端/mp小程序端/`
- 根目录 `mp-weixin/`，用于兼容旧部署文档和旧用户路径
