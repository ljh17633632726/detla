# 护航小程序

本目录是可直接部署的 Docker 交付包，不包含项目源码，包含后端、PC 管理后台、H5 页面、MySQL、Redis 镜像及一键部署文件。


##技术栈：Spring Boot、Vue 3、UniApp、微信小程序、微信支付、MySQL、Redis。

## 项目介绍

本系统适合用于搭建电竞陪玩、游戏代练、俱乐部护航、虚拟商品服务、同城/线上接单等平台。用户可以在小程序内浏览服务、提交订单、在线支付、查看进度并与客服或接单员沟通；接单员可以入驻、抢单、接单、组队、提交进度、查看收益并申请提现；客服和管理员可以在手机端或 PC 后台处理派单、退款、投诉、换人、提现和商品运营。

## 最近更新（2026-07-17）
- H5 页面已接入微信 H5 支付。
- 增加 Docker 离线镜像包和 Windows 一键部署能力。

- 增加余额充值套餐，支持充值金额和赠送金额配置。

## 核心能力

- 多端完整：用户小程序、接单员端、手机客服端、PC 管理后台。
- 订单闭环：下单、支付、派单、接单、进度、完成、评价、投诉、退款。
- 接单模式：客服派单、指定接单员、抢单、组队单、接力、换人。
- 运营管理：商品、分类、热门分类、动态下单字段、公告、轮播、快捷回复。
- 资金结算：微信支付、余额支付、退款、平台抽成、商品抽成、组队分成、提现审核。
- 风控配置：接单员押金、在线状态、并发接单上限、投诉处罚、禁止接单、禁止提现。
- 消息提醒：站内消息、聊天会话、订单卡片、商品卡片、阿里云短信提醒。

## Windows 一键部署

1. 下载 `delta-game-images.tar` 并放到本目录。
2. 复制 `.env.example` 为 `.env`。
3. 修改 `.env` 中的数据库密码、Redis 密码、JWT 密钥和 AES 密钥（默认参数也可以启动）。
4. 双击 `start.bat`，或使用 PowerShell 执行 `./start.ps1`。

启动后访问：

- 管理后台：`http://localhost:8081`
- H5 页面：`http://localhost:8082`
